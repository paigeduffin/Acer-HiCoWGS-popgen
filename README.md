# Acer-HiCoWGS-popgen
This repository includes scripts to recreate analyses for manuscript entitled "Limited neutral and adaptive genomic variation suggests Acropora cervicornis can be managed as a single conservation unit across its range" currently in preparation.

Raw sequencing reads generated in this study can be downloaded from the NCBI SRA database under BioProject PRJNA1094960. Previously sequenced samples from Panama (Vollmer et al 2023) can be sourced from NCBI BioProj PRJNA950067.
